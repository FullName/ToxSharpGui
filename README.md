ToxSharpGui
===========

Tox#GUI, a GUI for tox written in C#/MonoDevelop

Very early alpha stage, though a number of things already work.
It requires ToxSharpGuiGTK.dll or ToxSharpGuiWinforms.dll for the actual GUI.
